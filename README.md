#### Especificações do projeto:

![Pdf Da Atividade](https://i.imgur.com/4e8zs9m.jpeg).

#### Para acessar o projeto online clique no link abaixo:
- [Animal - Hotels](https://animal-hotels-taupe.vercel.app/)

#### Para rodar o projeto localmente siga os passos abaixo:

1 - Clone o repositório:
```bash
  git https://github.com/LincolnMatheus97/animal-hotels.git
```

2 - Para inicializar o projeto: 
```bash
  npm install
```
3 - Para rodar localmente o projeto, abra dois terminais e em um rode:

```bash
  npm run dev -> O frontend rodará 
  npm run api -> O backend rodará
```
